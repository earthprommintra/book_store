Book Store
------------
ระบบจัดการร้านหนังสือออนไลน์ที่พัฒนาด้วย PHP และ MySQL พร้อม Admin Panel

Demo : https://www.comsci-rmutp.com/2568gp23/products.php

Features
---------------------------------------

User Features

สมัครสมาชิกและเข้าสู่ระบบ,
เรียกดูรายการหนังสือ,
เพิ่มสินค้าลงตะกร้า,
จัดการตะกร้าสินค้า (เพิ่ม/ลด/ลบ),
ชำระเงิน (Cash on Delivery),
ตรวจสอบสถานะคำสั่งซื้อ,
ยกเลิกคำสั่งซื้อ และ
จัดการโปรไฟล์และที่อยู่จัดส่ง

Admin Features

Dashboard ภาพรวมระบบ,
จัดการผู้ใช้ (เพิ่ม/แก้ไข/ลบ),
จัดการสินค้า (เพิ่ม/แก้ไข/ลบ/อัพโหลดรูปภาพ),
จัดการคำสั่งซื้อ (ดูรายละเอียด/เปลี่ยนสถานะ) และ
รายงานสรุปยอดขาย

Tech Stack
---------------------------------------------

Backend: PHP 7.4+

Database: MySQL

Frontend: HTML5, CSS3, Bootstrap
